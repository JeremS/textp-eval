

# Textp-doc
This library provides some helper one may use to eval textp documents.

## Installation
Deps coords:
```clojure
#:fr.jeremyschoffen.textp{eval-alpha #:mvn{:version "0"}}
```
Lein coords:
```clojure
[fr.jeremyschoffen.textp/eval-alpha "0"]
```
